Full-Stack-AI--Job-Recommendation-System

Full-Stack-AI--Job-Recommendation-System is an intelligent AI-powered career recommendation platform designed to help students, fresh graduates, and job seekers discover suitable career opportunities based on their skills, qualifications, interests, and resume data. The platform combines Artificial Intelligence, Machine Learning, and Full Stack Web Development to deliver personalized job and internship recommendations.

🌟 Overview

The system analyzes resumes, skills, educational background, and user preferences to recommend relevant job roles and internship opportunities. It streamlines the recruitment and career guidance process by using AI-driven prediction models and smart recommendation algorithms.

The platform is built to bridge the gap between job seekers and recruiters while helping users make informed career decisions through data-driven insights and intelligent automation.

📦 Installation & Setup
Clone the repository:
git clone https://github.com/palak26022005/-Full-Stack-AI--Job-Recommendation-System.git
Move the project folder to the XAMPP htdocs directory.
Start Apache and MySQL using XAMPP Control Panel.
Import the project database into phpMyAdmin.
Install required Python dependencies for AI modules.
Run the project in your browser:
http://localhost/careerproject/
